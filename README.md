# C++ OOP Learning Programs

A collection of C++ programs covering core Object-Oriented Programming concepts, written as part of a learning journey through classes, inheritance, encapsulation, and more.

---

##  File Overview

| File | Topic |
|------|-------|
| `Add.cpp` | Basic addition using functions |
| `functiontonumber.cpp` | Function returning a value |
| `Ini_Test.cpp` | Compiler setup test |
| `Assignment_String.cpp` | String operations and concatenation |
| `Class01.cpp` | Introduction to classes |
| `Classes02.cpp` | Class with multiple objects (Car) |
| `Class03.cpp` | Class with multiple objects (Student) |
| `Prog_Stud.cpp` | Student class with user input |
| `Inline.cpp` | Passing objects as arguments (inline member function) |
| `NonInline.cpp` | Passing objects as arguments (non-inline member function) |
| `inline_func.cpp` | Standalone inline function example |
| `BankAccount.cpp` | Encapsulation, static members, deposit/withdraw logic |
| `Bank_withd.cpp` | Nested classes, inheritance with BankAccount |
| `friendClass.cpp` | Friend class accessing private members |
| `Inheritence.cpp` | Single inheritance (Vehicle → Car) |
| `Inhititence_pvt.cpp` | Protected members in inheritance |
| `7_4_INHERITENCE_class.cpp` | Single, multilevel, and multiple inheritance |

---

##  Concepts Covered

- **Classes & Objects** — defining blueprints and creating instances
- **Encapsulation** — using `private`/`public`/`protected` access specifiers
- **Inline & Non-Inline Functions** — member function definition styles
- **Static Members** — shared data across all objects
- **Friend Classes** — granting access to private members
- **Nested Classes** — classes defined inside other classes
- **Inheritance** — single, multilevel, and multiple inheritance
- **Constructors** — parameterized constructors and initializer lists

## 🛠️ How to Compile & Run

You need a C++ compiler such as `g++`.

```bash
g++ filename.cpp -o output
./output
```

Example:
```bash
g++ BankAccount.cpp -o bank
./bank
```
